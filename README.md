## Features - 專案功能

  1. 商家網頁版型設計
  2. 表單驗證
  3. AJAX頁面切換
  4. RWD 響應式網頁設計

## Technical Skills - 使用哪些技術實作專案

- HTML - 標籤語法運用
- CSS + SCSS - 規劃響應式網頁版型
- JavaScript - 表單內容驗證 ( Regex ) 
- jQuery - 製作動畫效果 ( Toggle Menu、Navbar )
- Ajax - 非同步切換選單 ( Tab panel )
